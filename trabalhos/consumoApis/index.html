Consumo de APIs - BrasilAPI

Abra o console do navegador (F12) para ver os dados completos retornados por cada API.

1. Consulta de CEP

Endpoint: GET https://brasilapi.com.br/api/cep/v1/{cep}

2. Lista de Bancos

Endpoint: GET https://brasilapi.com.br/api/banks/v1

3. Consulta de DDD

Endpoint: GET https://brasilapi.com.br/api/ddd/v1/{ddd}

4. Feriados Nacionais

Endpoint: GET https://brasilapi.com.br/api/feriados/v1/{ano}

5. Cidades (CPTEC)

Endpoint: GET https://brasilapi.com.br/api/cptec/v1/cidade/{nome}

6. Consulta de CNPJ

Endpoint: GET https://brasilapi.com.br/api/cnpj/v1/{cnpj}
